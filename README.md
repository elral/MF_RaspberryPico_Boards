## Firmware for RP2040 boards
Firmware with all devices for a self made board or other boards (e.g. PGA2040) where all pins from a RP2040 are available.

### Suppported Boards
* generic RP2040 board
* RP2040zero
* generic RP2350A board
* generic RP2350B board
* Pimoroni PGA2350
* Weact PGA2350B

### Installation
Unpack the latest released zip file and copy the content to your community folder inside the MobiFligth installation folder.
